二叉树的序列化与反序列化-题目描述

请实现两个函数，分别用来序列化和反序列化二叉树。

您需要确保二叉树可以序列化为字符串，并且可以将此字符串反序列化为原始树结构。

样例
你可以序列化如下的二叉树
8
/ \
12 2
/ \
6 4

为：”[8, 12, 2, null, null, 6, 4, null, null, null, null]”
注意:

以上的格式是AcWing序列化二叉树的方式，你不必一定按照此格式，所以可以设计出一些新的构造方式

中二叉树的序列化与反序列化-总体思路
